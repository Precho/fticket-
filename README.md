1.**Clone the repo**

```
$ git clone git@github.com:Precho/fticket.git
$ cd todo
```
2.**Install dependencies**

```
$ npm install

```
3.**Start Packager**

```
$ npm start
```
4.**Running on iOS:**

```
$ npm run-ios
```
5.**Running on Android:**

```
$ npm run-android
```
